ANG HR 0603｜Logo 切換員工端修正版

本包以「平台 Creator 四信兩驗證版」為底，並修改 Admin 頁 FAB / Logo 行為。

規則：
1. 右下 Home FAB 只負責展開 / 收合 L 型功能導航。
2. FAB 未展開時，點左上公司 Logo = 原本公司切換選單。
3. FAB 已展開時，點左上公司 Logo = 收合 FAB 並跳回 employee.html。
4. 展開後點其他地方仍會自動收回。
5. FAB 展開時 Logo 會稍微發亮，提示可點擊回員工端。

覆蓋：
- admin.html
- employee.html
- index.html
- config.js
- ang-frontend-api.js
- 程式碼.js
